# Moment Booth

A minimal React photobooth app for capturing 1-4 photos and exporting them as a clean photo frame.

## Features

- Camera capture with countdown
- Photo count options from 1 to 4
- Frame layouts such as 2x2, 1x4, and 4x1
- Minimal filter presets
- Optional date stamp
- Light and dark export themes

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
